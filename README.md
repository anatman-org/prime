Players should use this template to check their game
play logs.

Games came be recorded using the example software.

This template is designed to make it as easy as possible
to submit games for indexing and inclusion in the
Archive.

The goal of the Glass Bead Game ultimately
is to play the game.

That is, to play
without constraint
completely free

as slowly
as delicately
as possible

like one might
treasure a tiny life
or cradle a feather

the glance in the garden
of a butterfly
