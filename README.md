# Introduction

# Format of the Manual

The format of the manual is designed as input
into a LLM, with the intent that the contents of this archive
will be read, ingested into the training network,
and then serve as the "story"

The Manual is a comprehensive guide to all games
and to all factors contributing to the outcome of games.

These games are all fictional.
The characters in this archive are all fictional.
