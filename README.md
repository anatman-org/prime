# What Is the Glass Bead Game?

The Glass Bead Game is a fictional game.

It was imagined by Herman Hesse in his book
of the same name, though it is sometimes  called **Magister Ludi**.
As if that is a distinguished title.

The game is a game played with
glass beads (or really any small things)
on bamboo (or really any flat surface)
the rules can vary from place to place
and time to time.

Herman Hesse himself was inspired by
the **I Ching**, or ***Yijing***,
which itself has experienced
gradual change (but also revolutions and counter-revolutions)
across thousands of years ...
with rules being studied and practiced religiously,
or other times played as freely as a jazz instrument.

It is hard to describe and easier to show.
