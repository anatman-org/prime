# The Annotated Manual to the Glass Bead Game

lalla says,

    to learn the game is easy,
    to live it, hard.
    the search for the Real
    is no simple matter

    deep in my looking,
    the last words vanished.
    joyous and silent,
    the waking that met me there

the glass bead game is an investigation into reality

in a glass bead game,
there is nothing to hide

the mind is exposed in every way
vulnerable,
with every dark thought manifest in
what might otherwise be random noise

and with one's own two hands
one makes and remakes
the stories of existence
